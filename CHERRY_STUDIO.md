# Cherry Studio 配置指南

> **声明：本项目完全由 AI 生成，仅供个人学习研究使用，拒绝任何商业行为和非法行为。**

本文档详细介绍如何在 Cherry Studio 中配置和使用 DeepSeek API。

---

## 一、安装 Cherry Studio

下载地址：https://cherry-ai.com/

支持 Windows、macOS、Linux。

---

## 二、启动 API 服务

在配置 Cherry Studio 之前，需要先启动 API 服务。

```bash
# 进入项目目录
cd deepseek-reverse

# 配置登录信息（首次使用）
cp .env.example .env
# 编辑 .env 文件，填入 Token 或邮箱密码

# 启动服务
./start.sh
```

**保持终端窗口打开，不要关闭！**

---

## 三、配置 Cherry Studio

### 步骤 1: 打开设置

打开 Cherry Studio，点击左下角的齿轮图标 ⚙️。

### 步骤 2: 进入模型服务商

在左侧菜单中点击 **模型服务商**。

### 步骤 3: 添加供应商

点击页面上的 **添加** 按钮。

### 步骤 4: 选择供应商类型

选择 **OpenAI**。

### 步骤 5: 填写配置

| 配置项 | 填写内容 | 说明 |
|--------|----------|------|
| 供应商类型 | OpenAI | 固定选择 |
| 名称 | DeepSeek Local | 自定义名称 |
| API 密钥 | 留空 | 不需要填写 |
| API 地址 | `http://127.0.0.1:8000/v1` | **必须包含 /v1** |

### 步骤 6: 添加模型

在 **模型** 部分，点击 **添加** 按钮，输入模型名称：

| 模型名称 | 说明 |
|----------|------|
| `deepseek-chat` | 快速模式 |
| `deepseek-expert` | 专家模式 |
| `deepseek-chat-think` | 快速 + 深度思考 |
| `deepseek-chat-search` | 快速 + 智能搜索 |

### 步骤 7: 测试连接

点击 **测试** 按钮，显示成功即可。

### 步骤 8: 保存

点击 **保存** 按钮。

---

## 四、使用方法

1. 点击左侧 **+** 按钮创建新对话
2. 在顶部模型下拉菜单中选择 **DeepSeek Local** → 具体模型
3. 输入问题开始对话

### 上传附件

点击输入框左侧的 📎 图标，选择文件上传。

### 切换模型

在对话页面顶部的下拉菜单中，可以随时切换模型。

---

## 五、常见问题

### Q: 测试连接失败

**A:** 检查：
1. API 服务是否已启动
2. API 地址是否正确（`http://127.0.0.1:8000/v1`）
3. 端口 8000 是否被占用

### Q: 发送消息后没有响应

**A:** 可能原因：
1. Token 过期，需要重新获取
2. 模型名称错误

### Q: 如何使用邮箱密码登录？

**A:** 在 `.env` 文件中配置：
```
DEEPSEEK_EMAIL=your_email@example.com
DEEPSEEK_PASSWORD=your_password
```
